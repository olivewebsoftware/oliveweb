Welcome to OliveWeb's Official Discord Server!

This is a server to talk about OliveWeb's products, and
suggest some new software as well!


But, like every other Discord server, there are some rules
you have to follow as well:


1. No spamming

Spamming will result in a mute or kick, but it depends
how much you do it.


2. No malware

Spreading malware is a violation of the OliveWeb TOS.
This will lead to a permanent ban with no appeals.


3. Be kind

We don't want you to be mean, so you'll just get muted
for 10 minutes for being rude but the time may increase
depending on how you are.


4. Don't be annoying

Please don't annoy members on the server with annoying
things. Doing this a lot can get you kicked from the server.


5. No asking stupid questions

For example: "Can you release a free Bitcoin miner?"
Questions are welcome, just don't ask stupid ones over and
over again.


6. TOS Violations

If you are caught violating the OliveWeb TOS, your account
will get immediately with no appeals.


If you need any help with OliveWeb software or with the server
itself, contact us at customer-support.
